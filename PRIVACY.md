# PhishGuard 개인정보처리방침

**최종 수정일:** 2026년 2월 18일

PhishGuard(이하 "본 확장 프로그램")는 사용자의 개인정보를 중요하게 생각합니다. 본 개인정보처리방침은 본 확장 프로그램이 어떤 데이터를 수집·처리하며, 어떻게 보호하는지 설명합니다.

## 1. 수집하는 데이터

본 확장 프로그램은 피싱 탐지 기능을 수행하기 위해 다음 데이터를 처리합니다.

| 데이터 | 처리 방식 | 외부 전송 여부 |
|--------|----------|--------------|
| 방문 페이지 URL 및 도메인 | 브라우저 내 로컬 분석 | 전송하지 않음 |
| 페이지 HTML 콘텐츠 (제목, 폼, 링크 등) | 브라우저 내 로컬 분석 | 전송하지 않음 |
| 사용자 설정 (활성화 상태, API 키) | chrome.storage에 저장 | 전송하지 않음 |

## 2. 외부 서비스 통신

본 확장 프로그램은 다음 외부 서비스와 통신할 수 있습니다.

### 공공데이터포털 API (api.odcloud.kr)
- **목적:** KISA 피싱사이트 블랙리스트 데이터를 다운로드하기 위함
- **전송 데이터:** API 인증키, 페이지네이션 파라미터만 전송
- **사용자 데이터 전송:** 없음 (피싱 URL 목록을 수신만 함)

### LLM API (선택적 — 사용자가 직접 활성화한 경우에만)
- **목적:** AI를 활용한 피싱 사이트 정밀 분석
- **전송 데이터:** 분석 대상 페이지의 URL, 도메인, 콘텐츠 요약
- **조건:** 사용자가 설정에서 AI 분석을 활성화하고 직접 API 키를 입력한 경우에만 작동
- **제공자:** 사용자가 선택한 LLM 제공자 (Gemini, GLM 등)

## 3. 수집하지 않는 데이터

본 확장 프로그램은 다음 데이터를 수집하지 않습니다.

- 개인 식별 정보 (이름, 이메일, 전화번호 등)
- 로그인 자격 증명 (아이디, 비밀번호)
- 금융 정보 (카드번호, 계좌번호)
- 브라우징 히스토리 (방문 기록을 외부로 전송하거나 영구 저장하지 않음)
- 쿠키 또는 추적 데이터
- 위치 정보

## 4. 데이터 저장

- 분석 결과는 브라우저 세션 스토리지에 임시 저장되며, 브라우저 종료 시 삭제됩니다.
- KISA 피싱 블랙리스트는 chrome.storage.local에 캐싱되며, 확장 프로그램 제거 시 삭제됩니다.
- 사용자 설정은 chrome.storage.sync에 저장되며, 확장 프로그램 제거 시 삭제됩니다.

## 5. 제3자 제공

본 확장 프로그램은 사용자의 데이터를 제3자에게 판매, 거래 또는 제공하지 않습니다.

## 6. 데이터 삭제

사용자는 언제든지 확장 프로그램을 제거하여 저장된 모든 데이터를 삭제할 수 있습니다. 설정 초기화를 통해 저장된 API 키 및 설정을 개별 삭제할 수도 있습니다.

## 7. 변경 사항

본 개인정보처리방침이 변경되는 경우, 이 페이지에 업데이트됩니다.
